The site went live today.

Yesterday the code existed on my laptop. Today it's at insideai.in — SSL padlock, custom domain, auto-deploying from GitHub every time I push a change.

Here's what happened:

Deployed to Netlify in 90 seconds. Updated two DNS records in GoDaddy. Waited 20 minutes for propagation. The padlock appeared.

Set up Beehiiv for the newsletter. Subscribe form is now live on every page. Readers enter their email, go straight into my Beehiiv account, get posts by email. Free up to 2,500 subscribers.

Set up Giscus for comments. Readers sign in with GitHub. No spam bots, no moderation needed.

Submitted to Google Search Console. Google now knows insideai.in exists.

Then I wrote my first article — directly on GitHub, in the browser. Just a text file with some formatting at the top. Committed it. Netlify built the site in 60 seconds. Article was live.

Then I wrote a second one. A complete guide on how to build a blog with AI in one day. Every tool, every step, every mistake.

What keeps surprising me: none of this required me to understand how it works. I know what each tool does. I don't know how. Claude handles the how.

That's a new way to build things. I'm still adjusting to it.

Tomorrow: write more. That's the only job now.
